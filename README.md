# Facial Emotion Recognition

Real-time Facial Emotion Recognition using OpenCV and TensorFlow.

## Features

- Face Detection
- Emotion Classification
- Real-time Webcam Prediction
- FER2013 Dataset Support

## Emotions

- Angry
- Disgust
- Fear
- Happy
- Sad
- Surprise
- Neutral

## Installation

```bash
pip install -r requirements.txt
```

## Training

```bash
python train.py
```

## Run Detection

```bash
python detect.py
```

## Dataset

Download FER2013 dataset and place it in:

```text
dataset/fer2013.csv
```
